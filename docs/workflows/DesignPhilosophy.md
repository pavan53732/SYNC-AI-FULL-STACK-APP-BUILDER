# Design Philosophy

> **User Workflows: Guiding Principles and UX Values**
>
> **Parent Document:** [USER_WORKFLOWS.md](../USER_WORKFLOWS.md)

---

## Guiding Principles

1. **AI as Primary Brain**: The user describes intent, AI handles all technical execution.
2. **Local-First Privacy**: All code and data stays on the user's machine.
3. **No IDE Required**: Zero exposure to development tooling.
4. **Silent Resilience**: The system auto-recovers from errors without user intervention.
5. **Progressive Disclosure**: Advanced features hidden until needed.

---

## UX Values

- **Calm**: No red errors, no stack traces, no technical jargon.
- **Fast**: Sub-second preview updates, responsive UI.
- **Safe**: Every change is snapshot, every action is reversible.
- **Honest**: When something fails, explain in plain language.
