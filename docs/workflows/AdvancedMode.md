## Developer Mode

For users comfortable with technical details:

| Hidden by Default      | Visible in Dev Mode                     |
| :--------------------- | :------------------------------------- |
| Orchestrator state     | Real-time state transitions             |
| Build errors           | Raw MSBuild output (optional)           |
| Retry attempts         | Retry counter and backoff timing        |
| Task names             | Full task graph visualization           |
| AI prompts             | Prompt Inspector window                 |

---

## Enable Developer Mode

Settings → Developer Mode → Enable

This unlocks:
- BuildMonitorPanel with full orchestrator visualization
- Raw log viewer
- Task graph inspector
- Prompt history viewer
